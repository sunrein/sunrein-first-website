Enumerable Method - Map 11/1/2015
What it does, how it works

When writing this blog post, I was very interested in looking at the method .map. I have seen it used a few times, but was never sure what it was doing. I don't think the fact that it's name doesn't as bluntly refer to what action it performs (like .each or .times, for instance). .map is a method that performs an action to each element in a data set, and .map! can be used destructively to permanently alter the data structure (while of course, .map, does not do this). It is the exact same method as .collect, whose name I also find confusing. Neither of them allude directly to the fact that they alter a data set. I read that .map is more commonly used, so I'll stick with that one. A helpful website I found said to think of it like you're "remapping" the data.
